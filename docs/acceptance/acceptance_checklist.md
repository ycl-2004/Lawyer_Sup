# 验收 Checklist

> 验收人无需提问即可逐项独立验证。每项列出**操作 → 预期结果**，勾选即通过。
> 全部为虚构演示数据；真实/模拟边界见 [demo_script.md](./demo_script.md) 诚实声明页。

## A. 环境与启动

- [ ] **A1 一键启动**：双击 `start_mac.command`（Win：`start_windows.bat`）→ 浏览器自动打开
  `http://localhost:8000`，看到 Dashboard。（首次需联网装依赖+构建，2–5 分钟）
- [ ] **A2 开发模式**：`npm install && npm run dev`（前端 5173）；另开终端 `cd backend && pip install -r requirements.txt && uvicorn app.main:app --port 8000`。
- [ ] **A3 健康检查**：访问 `http://localhost:8000/api/health` → `{"status":"ok","llm_provider":"mock",...}`。
- [ ] **A4 接口文档**：访问 `http://localhost:8000/docs` → FastAPI Swagger 列出全部接口。

## B. 实时案件全链路（核心，全新「未见过」案件）

- [ ] **B1 新建案件**：Dashboard →「新建案件」→ 填当事人/对方 → 进入空工作台。
- [ ] **B2 上传 PDF**：上传一份含「甲方/乙方/月工资…元/于…解除劳动合同」的劳动合同或解除通知 PDF
  → 材料列表出现该件，状态「已解析」。
- [ ] **B3 解析失败兜底**：上传扫描件/图片 → 提示解析失败 → 点该材料「手动粘贴文本」录入 → 状态转「已解析」。
- [ ] **B4 运行工作流**：「运行 Agent 工作流」→ 弹窗显示逐节点（解析→分类→抽取→时间线→请求项→计算→RAG→草稿→复核）绿色完成，带「后端管线 · FastAPI」徽章。
- [ ] **B5 真实抽取**：工作区出现抽取事实（当事人/入职日期/月工资/解除日期等），每项带**原文来源引用**。
- [ ] **B6 持久化**：**刷新浏览器** → 案件与全部抽取数据仍在；**重启后端**后再访问 → 数据仍在。
- [ ] **B7 确认审计**：复核页填律师姓名 →「确认复核完成」→ 显示确认人 + 时间戳（重启后仍在）。

## C. 「不许猜」红线（最具说服力的演示）

- [ ] **C1 计算拒绝**：刘某案（加班费）加班小时为空 → 计算卡片显示**「无法计算 — 缺少输入」**而非编造数字；手动填入小时数后立即算出金额。
- [ ] **C2 缺输入拒绝**：经济补偿缺月工资 → 卡片拒绝计算并列出缺失项。
- [ ] **C3 日期冲突拒绝**：解除日期早于入职日期 → 拒绝计算并提示「日期冲突」。
- [ ] **C4 引用反幻觉**：复核页「核对全部草稿引用」→ 含《不存在法》或《劳动合同法》第九百条等
  幻觉引用被标为**高风险未命中**；真实存在的条文命中并显示生效日期。
- [ ] **C5 导出门槛**：未处理高风险发现 / 未勾选确认清单 / 未填律师姓名时，「确认复核完成」按钮**禁用**，导出不可点。

## D. 演示案件与导出

- [ ] **D1 张某案**：默认载入 → 概览/材料预览（点行看全文+分类依据）/时间线/请求与计算（改月工资看实时重算）。
- [ ] **D2 docx 导出**：文书页导出 docx → 用 Word 打开，**首页有红色「草稿」水印**。
- [ ] **D3 材料包导出**：复核确认后导出 Markdown 材料包 → 含免责声明、确认审计记录、占位符计数、脱敏兜底。

## E. 测试与质量（命令行）

- [ ] **E1 前端测试**：`npm test` → 41 passed；`npm run build` → 类型检查 + 构建成功。
- [ ] **E2 后端测试**：`cd backend && pytest` → 50 passed。
- [ ] **E3 抽取评估**：`python -m app.eval.run_eval` → 召回 84% / 精确 100% / **编造 0** / **引用有效 100%**。
- [ ] **E4 语料一致性**：`python -m app.rag.verify_corpus` → 5/5 校验通过。
- [ ] **E5 测试存档**：对照 [test_output_archive.md](./test_output_archive.md) 复现一致。
